# Packaging
This directory contains all the packaging resources necessary to package BeatSketch.
Supported:
- Pacman (Arch Linux): PKGBUILDs provided (also fully built package in releases)
- RPM (RedHat, Fedora, etc): rpm package provided in the releases, spec file provided
- DEB (Debian, Ubuntu, etc): Will be provided in the future
- Windows: Installer or simple archive download with built application
- Mac: Will be provided in the future

For Arch, we are also providing also a bleeding edge (-git) PKGBUILD.
Both PKGBUILDs are available in the Arch User Repository (AUR) as [beatsketch](https://aur.archlinux.org/packages/beatsketch)
and [beatsketch-git](https://aur.archlinux.org/packages/beatsketch-git)
